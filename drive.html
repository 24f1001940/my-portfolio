    About Drive Explorer

    Drive Explorer v2.0

    An advanced Google Drive public folder browser with powerful features for exploring and managing your files.

    Features:

    • Three-pane responsive layout
    • Real-time search and filtering
    • File preview and details
    • Keyboard shortcuts
    • Dark/light theme support
    • Virtual scrolling for performance
    • Context menu actions
    • Drag and drop support

    Credits:

    Built with vanilla JavaScript, CSS Grid, and the Google Drive API v3.
